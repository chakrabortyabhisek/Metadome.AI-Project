AI Home Decor Room – Unity Project

Interactive room design experience with AI-powered decor suggestions.

Overview

This project demonstrates a simple **interactive home decor environment** built using **Unity**.
Users can place furniture, change wall colors, and receive **AI-generated decor suggestions** based on design styles.

This project was developed as part of a **Solutions Engineer evaluation assignment**.

---

 Features

 Furniture Placement

Users can place furniture items such as:

* Sofa
* Chair
* Table
* Lamp

Furniture can be dynamically added to the room using UI controls.

---

 Wall Color Customization

Users can change wall colors in real time:

* White
* Blue
* Green

This provides instant visual feedback.

---

 AI Decor Suggestions

Users select a style:

* Modern
* Minimalist
* Luxury

The system generates **3 decor suggestions** for furniture arrangement and color themes.

Example output:

1. Grey sofa with glass table
2. Neutral wall colors with wooden furniture
3. Floor lamp with minimalist decor

---
 Generative AI Component

The system simulates generative AI recommendations by generating style-based decor suggestions.

Architecture:

User Input → Style Selection → AI Suggestion System → Decor Recommendations

---

 Controls

| Action        | Description                |
| ------------- | -------------------------- |
| Add Sofa      | Place sofa in the room     |
| Add Chair     | Place chair                |
| Add Table     | Place table                |
| Wall Buttons  | Change wall color          |
| Style Buttons | Generate decor suggestions |

 How to Run

1. Clone the repository

```
git clone https://github.com/yourusername/ai-home-decor-unity
```

2. Open project in **Unity Hub**

3. Open scene:

```
Assets/Scenes/MainScene
```

4. Press **Play**
